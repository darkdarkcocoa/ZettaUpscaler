🔧 문제 해결
─────────────────────────
문제: CUDA not available
해결: .\.venv\Scripts\pip install torch==2.2.0+cu121 torchvision==0.17.0+cu121 --index-url https://download.pytorch.org/whl/cu121

문제: BasicSR import error
해결: python fix_compatibility.py

문제: NumPy 버전 충돌
해결: .\.venv\Scripts\pip install numpy==1.26.4
